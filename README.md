🏥 Clinic Patient Management System
📌 Project Overview

The Clinic Patient Management System is a Java-based application developed using Hibernate ORM for efficient database interaction.
This system helps manage patient records and appointment scheduling in a clinic environment.

It provides basic CRUD operations for patients and supports appointment management through a console-based interface.

🚀 Features

✅ Add new patient records

✅ View patient details

✅ Update patient information

✅ Delete patient records

✅ Book appointment

✅ Cancel appointment

✅ Hibernate-based database connectivity

🛠️ Technologies Used

Java

Hibernate ORM

Maven
